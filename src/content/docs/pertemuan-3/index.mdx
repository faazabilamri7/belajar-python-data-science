## ğŸš€ Quick Start

```python
import pandas as pd
import numpy as np

# NumPy: create array
arr = np.array([1, 2, 3, 4, 5])
print(arr.mean())  # 3.0

# Pandas: create DataFrame
df = pd.DataFrame({
    'nama': ['Andi', 'Budi', 'Citra'],
    'nilai': [85, 90, 78]
})
print(df.describe())  # statistik
```

## âœï¸ Latihan Cepat

Coba jalankan code ini di Google Colab:

```python
import pandas as pd
import numpy as np

# NumPy
print("=== NumPy ===")
arr = np.array([1, 2, 3, 4, 5])
print(f"Array: {arr}")
print(f"Mean: {arr.mean()}")
print(f"Sum: {arr.sum()}")

# Pandas
print("\n=== Pandas ===")
data = {
    'nama': ['Faaza', 'Budi', 'Citra'],
    'umur': [25, 23, 22],
    'nilai': [85, 90, 88]
}
df = pd.DataFrame(data)
print(df)
print(f"\nRata-rata nilai: {df['nilai'].mean()}")
```
