## ğŸ§  Skill yang Dibutuhkan

### Hard Skills (Technical)

#### 1. Programming
- **Python** (essential)
- **SQL** (wajib)
- **R** (optional, untuk stats)
- Git/Version control

**Learning Path:**
```
Week 1-2: Python basics
Week 3-4: Data structures, OOP
Week 5-6: SQL queries
Week 7+: Advanced Python features
```

#### 2. Mathematics & Statistics
- **Linear Algebra** - Matrices, vectors untuk ML
- **Calculus** - Optimization, gradients
- **Probability** - Distributions, Bayes theorem
- **Statistics** - Hypothesis testing, correlation

**Minimal Knowledge:**
- Mean, median, variance, standard deviation
- Normal distribution, confidence intervals
- Hypothesis testing basics
- Correlation vs causation

#### 3. Data Processing
- **EDA (Exploratory Data Analysis)** - Understand your data
- **Data Cleaning** - Handle missing values, outliers
- **Feature Engineering** - Create meaningful features
- **Data Preprocessing** - Normalize, scale, encode

#### 4. Machine Learning
- **Supervised Learning** - Regression, Classification
- **Unsupervised Learning** - Clustering, Dimensionality Reduction
- **Model Evaluation** - Metrics, cross-validation
- **Hyperparameter Tuning** - Optimize model

**Common Algorithms to Know:**
- Linear Regression
- Logistic Regression
- Decision Trees
- Random Forest
- K-Means Clustering
- SVM

#### 5. Tools & Platforms
- **Jupyter Notebook** - Interactive development
- **Google Colab** - Cloud-based notebook
- **SQL Database** - PostgreSQL, MySQL
- **Cloud Platforms** - AWS, Google Cloud, Azure
- **Data Visualization** - Tableau, Power BI (nice to have)

### Soft Skills (Non-Technical)

#### 1. Problem Solving
- Break complex problems into steps
- Think critically about data
- Ask the right questions
- Don't stop until you understand WHY

**Practice:**
```
Kaggle competitions
HackerRank challenges
Project-based learning
```

#### 2. Communication
- Explain technical concepts in simple terms
- Create clear visualizations
- Write good documentation
- Present findings to non-technical stakeholders

**Why Important?**
- Even brilliant analysis is useless if no one understands it
- Business decision makers often don't understand ML
- Your job is to translate data into business value

#### 3. Curiosity & Continuous Learning
- Data Science field evolves rapidly
- New algorithms, tools, techniques constantly emerge
- Read papers, follow blogs, take courses
- Experiment and play with data

**Stay Updated:**
- Follow top researchers on Twitter
- Subscribe to newsletters (Towards Data Science, etc.)
- Join communities (Kaggle, Reddit r/MachineLearning)
- Attend meetups and conferences

#### 4. Business Acumen
- Understand business context
- Know what metrics matter
- ROI, revenue impact, cost reduction
- Think like a business analyst, not just a data analyst

**Example:**
- ğŸ“Š Technical: "Model has 95% accuracy"
- ğŸ’¼ Business: "Model saves company $2M annually by preventing fraud"

#### 5. Attention to Detail
- Small bugs can lead to wrong conclusions
- Data quality is crucial
- Document everything
- Validate results multiple times
