### Anscombe's Quartet: The Power of Visualization

Empat dataset ini memiliki **statistik yang SAMA**:
- Mean X = 9.0, Mean Y = 7.5
- Variance X = 11, Variance Y = 4.1
- Correlation = 0.816

Tapi ketika divisualisasikan, **semua terlihat BERBEDA!**

![Anscombe's Quartet](https://upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Anscombe%27s_quartet_3.svg/800px-Anscombe%27s_quartet_3.svg.png)

**Lesson**: Statistik saja tidak cukup - **VISUALISASI ESSENTIAL untuk understand data!**

## ğŸ“š Matplotlib vs Seaborn

### Kapan Gunakan Masing-masing?

| Aspect | Matplotlib | Seaborn |
| ------ | ---------- | ------- |
| **Learning Curve** | Steeper (lebih verbose) | Gentler (simpler API) |
| **Customization** | Sangat flexible | Less flexible |
| **Default Aesthetics** | Basic/plain | Beautiful defaults |
| **Statistical Plots** | Perlu coding manual | Built-in (histplot, boxplot, etc) |
| **Pandas Integration** | Limited | Excellent |
| **Use Case** | Fine-grained control | Quick EDA & reporting |

### Best Practice

**Hybrid approach:**
- Use **Seaborn** untuk quick visualization & EDA
- Use **Matplotlib** untuk fine-tuning & customization
- Combine both untuk best results!

```python
# Seaborn plot base + Matplotlib customization
sns.scatterplot(data=df, x='x', y='y', hue='category')
plt.title('My Custom Title', fontsize=16, fontweight='bold')
plt.ylabel('Y Axis Label', fontsize=12)
```

## ğŸ¨ Visualization Principles (Gestalt Laws)

### 1. Proximity
Points positioned close together appear as a group

### 2. Similarity
Objects with similar colors/shapes appear related

### 3. Continuity
Eyes follow smooth curves/lines

### 4. Closure
Brain completes incomplete shapes

### 5. Figure-Ground
Ability to separate foreground (chart) from background

**Application in data viz:**
- Group related data dengan positioning
- Use color consistently (same color = same category)
- Make trend lines smooth
- Use grid sparingly (tidak overpower data)
