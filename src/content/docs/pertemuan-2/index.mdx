## âœï¸ Latihan Cepat

Coba jalankan code ini di Google Colab atau local environment:

```python
# Setup dasar
nama = "Faaza"
umur = 25

# Kondisional
if umur >= 18:
    print(f"{nama} sudah dewasa")
else:
    print(f"{nama} masih remaja")

# List dan loop
hobi = ["coding", "reading", "gaming"]
for item in hobi:
    print(f"Saya suka {item}")

# Function
def hitung_kuadrat(x):
    return x ** 2

hasil = hitung_kuadrat(5)
print(f"5 kuadrat = {hasil}")
```
