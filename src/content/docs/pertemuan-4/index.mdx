## ğŸš€ Quick Start

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# Load dataset
df = sns.load_dataset('titanic')

# 1. Inspeksi dasar
print(f"Shape: {df.shape}")
print(df.head())
print(df.info())

# 2. Cek missing
print(df.isnull().sum())

# 3. Statistik
print(df.describe())

# 4. Visualisasi
df.hist(figsize=(12, 10))
plt.show()
```

## âœï¸ Latihan Cepat

Coba jalankan ini di Google Colab:

```python
import pandas as pd
import seaborn as sns

# Load data
df = sns.load_dataset('titanic')

# 1. Berapa banyak rows dan columns?
print(f"Shape: {df.shape}")

# 2. Ada berapa missing values?
print(f"\nMissing values:\n{df.isnull().sum()}")

# 3. Statistik kolom numerik
print(f"\nStatistik:\n{df.describe()}")

# 4. Unique values per kategori
print(f"\nUnique values:")
print(f"Sex: {df['sex'].unique()}")
print(f"Embarked: {df['embarked'].unique()}")
```
